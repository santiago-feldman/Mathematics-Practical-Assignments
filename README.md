# Mathematics Practical Assignments | ITBA

This repository contains the practical assignments done for the following subjects at Instituto Tecnológico de Buenos Aires:

- **93.24 Probability and Statistics**
  - Descriptive Statistics (Octave script)
  - Point estimation and confidence intervals (Python script)
- **93.54 Numerical Methods** (MATLAB scripts)
  - Systems of linear equations (Gaussian elimination, LU, QR, Jacobi, Gauus-Seidel, SOR)
  - Cholesky decomposition
  - Linear regression
  - Bisection method
  - Fixed-point iteration and Newton-Raphson methods
